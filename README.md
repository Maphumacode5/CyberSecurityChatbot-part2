# CyberSecurityChatbot-part2
Cybersecurity Awareness Chatbot

Description

This project is a C# console-based chatbot designed to educate users about cybersecurity awareness, including password safety, phishing, malware, and safe browsing practices.

Features

Voice greeting using audio
Interactive chatbot responses
Intelligent keyword-based response system
ASCII user interface
Cybersecurity awareness education

How to Run

1. Open the project in Visual Studio
2. Build the solution (Ctrl + Shift + B)
3. Run the program (Ctrl + F5)
